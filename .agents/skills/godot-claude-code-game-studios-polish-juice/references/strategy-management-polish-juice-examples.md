# Strategy And Management Polish / Juice Examples

Use this reference after the core loop works and the feature needs clearer,
more satisfying, and more readable feedback.

## Polish Targets By Archetype

| Sample | Polish target | Best signals |
|---|---|---|
| Taiko Risshiden | Make menu-heavy roleplay feel like living in a world. | Event cadence, travel rhythm, relationship changes, rank-up ceremony, minigame result feedback. |
| Kairosoft-style tycoon | Make every small optimization feel productive. | Coin ticks, customer motion, satisfaction bubbles, upgrade sparkle, unlock chain, contest result. |
| Plague Inc. | Make abstract systemic change feel physical and escalating. | Bubble pop, map color bloom, news ticker, cure warning, country collapse, global counter motion. |

## Polish Pass Checklist

- Reward events are short, frequent, and varied.
- Warning events interrupt only when the player must respond.
- Numeric growth is visible at the point of cause.
- Agent movement or map spread shows the system working, not just the UI.
- Unlocks reveal a new decision, not only a badge.
- Fast-forward keeps critical events readable.
- Accessibility fallback exists for color, sound, and motion-only cues.

## Before / After Examples

| Flat version | Polished version |
|---|---|
| Income number changes in top bar. | Facility emits a small coin tick, local floating value, then top bar increments. |
| New region infected message appears. | Region pulses, map color spreads, infection counter jumps, alert sound plays once. |
| Rank increases silently. | Title badge appears, available commands highlight, NPC reaction or event text confirms social change. |
| Upgrade completes in a menu. | Facility animates, throughput indicator updates, new decision is highlighted. |

## Common Mistakes

- Adding more popups instead of better in-world or in-place feedback.
- Treating every income tick as a major reward.
- Making alerts visually strong but mechanically uninformative.
- Hiding why a combo, unlock, or outbreak happened.
- Polishing before the loop has a meaningful decision.
