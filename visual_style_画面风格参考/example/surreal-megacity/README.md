## 风格还原点

- **高密度俯瞰群像**:斜俯视构图,远景富士山 + 中景街道游行 + 近景楼房窗格,层层可读
- **窗格叙事**:每扇窗 / 每个店面是一个独立微叙事(社畜、情侣、宅男、舞女),用细节堆叙事而非堆乱
- **超现实符号**:巨型招财猫 / 能面气球 / 霓虹汉字招牌,真实城市混入超现实意象
- **冷紫夜色 + 暖红霓虹**:整体冷调压抑,局部暖光点睛,繁华与孤独对冲
- **审美优先**:细节服务于构图与情绪,不为堆量牺牲协调感
