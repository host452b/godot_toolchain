对应总览表 #5 [暗黑手绘风 / Dark Hand-drawn](../../README.md#1-2d-游戏典型画风总览20-种)。

> **图 1**:grimdark 风格的瘟疫医生。注意其清晰的黑色重描边和类似 Comic Book 的排线阴影,完美契合《暗黑地牢》式的压抑感。

## 风格还原点

- **重墨排线**:粗细变化明显的手绘墨线 + 交叉排线(crosshatching)塑造体积
- **极端高对比度阴影**:大面积浓重暗部 + 局部硬高光,剪影辨识度极强
- **压抑的低饱和冷色调**:墨绿 / 灰褐为主,仅用提灯的暖绿光作唯一焦点色
- **氛围光**:单一光源(提灯)制造戏剧性明暗与孤寂感

## 参考 Prompt

```
dark grimdark hand-drawn 2D character art, plague doctor with beaked mask and wide-brim hat,

```

**Negative**: bright cheerful colors, flat lighting, low contrast, photorealism, 3D render, soft pastel.
