## 风格还原点

- **次世代动态光影**:像素之上叠加现代引擎光照,体积光、动态阴影、地面反射
- **霓虹撞色(Pink & Cyan)**:暗背景 + 高亮粉 / 青霓虹,强对比
- **电影级容积雾**:雨雾 / 霓虹辉光制造纵深与氛围
- **保留像素网格**:细节极高但仍可见像素颗粒,复古与高级感并存
